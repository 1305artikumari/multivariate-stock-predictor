# Multivariate Stock Price Prediction
An end-to-end machine learning and deep learning project that predicts the next trading day's closing price for **Reliance Industries (RELIANCE.NS)** using historical OHLCV data and technical indicators.
## Live Dashboard
[Open the deployed Streamlit dashboard](https://multivariate-stock-predictor-c5yvw4bqwwpkao2rcu6pqb.streamlit.app/)
## Project Overview
This project uses multiple market and technical features instead of only the closing price. It includes the complete workflow:
- Historical stock-data loading and validation
- Exploratory data analysis
- Technical feature engineering
- Chronological train/validation/test split
- Multiple ML/DL models
- Model comparison and evaluation
- Next-day closing-price prediction
- Streamlit dashboard deployment
## Features Used
The project creates 45 model features, including:
- Open, High, Low, Close, Volume
- Daily and log returns
- High-Low range and Open-Close change
- Close, return, and volume lag features
- 5, 10, 20, and 50-day moving averages
- Rolling standard deviation and volatility
- RSI (Relative Strength Index)
- MACD and signal line
- Bollinger Bands
- ATR (Average True Range)
- Volume moving average and volume ratio
- Day of week and month
## Models Compared
| Model | Purpose |
|---|---|
| Naive Baseline | Predicts tomorrow's close as today's close |
| Linear Regression | Interpretable linear baseline |
| Random Forest | Captures nonlinear feature relationships |
| XGBoost | Best-performing model in this project |
| LSTM | Deep-learning sequence model |
## Evaluation Metrics
Models are evaluated using:
- MAE — Mean Absolute Error
- RMSE — Root Mean Squared Error
- R² Score
- MAPE — Mean Absolute Percentage Error
- Directional Accuracy
## Test Results
| Model | RMSE | R² | Directional Accuracy |
|---|---:|---:|---:|
| XGBoost | 18.22 | 0.972 | 50.75% |
| Naive Baseline | 18.23 | 0.972 | — |
| LSTM | 19.05 | 0.969 | 53.00% |
| Linear Regression | 19.29 | 0.969 | 52.25% |
| Random Forest | 26.81 | 0.939 | 48.25% |
XGBoost was selected as the best model based on the lowest test RMSE.
